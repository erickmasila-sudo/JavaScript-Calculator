### JavaScript Calculator

This is a project that takes in two numbers from the user and gives the output of the addtion, subtraction, division and multiplication of the 2 numbers. It also allows the user to track the history of their calculations

## Installation

Clone the repository git@github.com:erickmasila-sudo/JavaScript-Calculator.git

## Usage

To run the project, use the following code in the command propmpt: node calculator.js
**The Expected Output is**

 Would you like to see your current history, close the calculator or use the calculator\
 Current history: Press h\
 Close: Press c\
 Use: Press u\
*>>*

**1. Pressing h** will reveal the currrent history in the terminal, because we starting the program the expected output is\
*[]* \
because no calculations have been done 
but if you press h after using the calculator after some time it will display the array in this formart

[
  '4, 8  addition: 12 subtraction: -4 multiplication: 32 division: 0.5',\
  '9, 3  addition: 12 subtraction: 6 multiplication: 27 division: 3'\
]

**2. Pressing u** will cause the program to display

Opening Calculator...\
 Enter the First Number?
 
 An Example: Type "3"
 
 This will cause the program to display\
 Enter second number?
 
 An Example: Type "5"

**The Output:**\
addition: 8 subtraction: -2 multiplication: 15 division: 0.6 

**3. Presssing c**\
This will display the text *Closing calculator...* and closes the program.
